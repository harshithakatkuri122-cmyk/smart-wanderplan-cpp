# Smart WanderPlan – Travel Planning System (C++)

## Overview
Smart WanderPlan is a console-based travel planning system developed in C++.
It helps users plan national and international trips based on budget,
trip purpose, number of travelers, and preferences.

The system dynamically suggests destinations, transport options,
hotel accommodations, and itineraries.

## Key Features
- Customer detail validation (email, phone number)
- National & international destination selection
- Budget-based destination filtering
- Dynamic itinerary suggestions
- Currency conversion for international trips
- Transport booking simulation
- Hotel accommodation planning
- Family, couple, friends, and solo trip support

## Concepts & Technologies Used
- C++ (STL)
- Object-Oriented Programming (OOP)
- Data Structures:
  - Binary Tree (destination preference tree)
  - Map
  - Vector
- String processing & validation
- Modular system design

## How to Run
1. Compile the program:
   g++ main.cpp -o wanderplan
